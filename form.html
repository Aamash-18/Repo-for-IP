Form in HTML5

Syntax:
<form action="" method="" target="">
Attribute Description
action url of next page to which we want to submit data.
method method which will upload data. The common methods used are GET & POST
target set the name of target window/frame where we want to display result.
enctype specify the way to encode data before sending it to server.

Input in HTML5

Syntax:
<input type="" name=""/>
input tag types:
type Description example
text used for single line text input
password used for password input
checkbox used to give list of items for multiple selection
radio used to give list of items for single selection
file used to give option for file upload
submit used to submit a form
reset used to clear a form
button used to call a client-side script
