Forms in HTML5:

To create form in html we use <form>

Attribute Description
action url of next page to which we want to submit data.
method method which will upload data. The common methods used are GET & POST
target set the name of target window/frame where we want to display result.
enctype specify the way to encode data before sending it to server.

There are few types of inputs which we can use in html

Text Input Controls:
three types of input contols are used on form.
  1. single line input
    2. This control is used for one line for user input.For this type value will be text
  2. Password input
    3. This control is used for password. For this type value will be password
  3. Multi line text input
    4. This control is used to accept detailed information from user. It is created by using <textarea> tag.
input tag attributes
Attribute Description
type specifies the type of control.
name used to assign a name to control which is sent to the server to be recognized and retrieve the value.
value used to set default value to the control
size set the width of teh control in therms of characters.
manlength specifies maximum no of characters which user can give in the text box.
