Variables:

To declare a variable in Js we use "var" keyword which means variant.
See the first line of script in file.

Operators:

  1. Arithmetic Operators. +, -,*, / and %
  2. Increment and Decrement Operators. ++, -- (pre and post)
  3. Relational Operators. ==, !=, >, <, >=, <=
  4. Logical Operators. &&, ||, !,
  5. Bitwise Operators. &, |, ^, ~, <<, >>, >>>
  6. Conditional Operators or Ternary Operator.
