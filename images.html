Images in HTML5

To insert an image in web page we use "img" tag.

Syntax:
<img src="url/path" alt="" width= "" height="" border="" align="">
src:
It is used to specify the url of image or path of image where we store image. It can show .png .jpeg or .gif files on webpage
alt:
It specify the alternate text for the image if for any resone image can not display. It contain text value.
width:
It specify the width of an image. It is a number value.
height:
It specify the height of an image. It is a number value.
align:
it specify the alignment of image.Its value can be right/left(default)/center.
border:
It specify the border width of an image.
