List in Html5

In HTML5 there are three ways to give a list:

  1. An unordered list
  2. An ordered list
  3. A definition list

Unordered List:

  • In this type, plain bullets are used for the list items.
  • This option is used when there is no any standard sequence/order of list items.
  • To use this type of list we use ul tag.
  • In ul tag we use li tag for individual list elements.

Ordered List:

  • In this type, generally numbers are used for the list items.
  • To use this type of list we use ol tag.
  • In ol tag we use li tag for individual list elements.
  • type="" attribute is used to specify type of number for list items.
    • 1-Number
    • I-Upper Roman
    • i-Lower Roman
    • A-Upper Alpha
    • a-Lower Alpha
  • start="Number" attribute is used to specify the starting point of numbering.

Definition List:

  • We use this type when we need like dictionary or encyclopedia type list.
  • To use this type of list we use dl tag.
  • In dl tag we use dt tag for specify a term.
  • In dt tag we use dd tag for specify a term definition.
